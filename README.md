# Executive Corporate Landing Page

## 📌 Project Overview
This project is a fully responsive corporate landing page developed using HTML, CSS, and JavaScript.  

The objective of this project is to demonstrate frontend development skills, including responsive layout design, modern UI styling, and client-side data handling using browser storage.

This project was developed as part of a technical evaluation task.

---

## 🚀 Features

- Modern corporate UI design
- Responsive layout (Desktop + Mobile)
- Navigation menu toggle for smaller screens
- Contact form with validation
- Data persistence using browser localStorage
- Dynamic rendering of submitted messages
- Clean and structured codebase

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (ES6)
- Browser localStorage API

## 💾 Data Storage Approach

Since this project is frontend-only, form submissions are stored using the browser's localStorage API.  

Data is:
- Stored in JSON format
- Persisted across page refresh
- Rendered dynamically on page load

This simulates basic client-side data persistence without a backend.

## 📱 Responsive Design

The layout is fully responsive and adapts to:
